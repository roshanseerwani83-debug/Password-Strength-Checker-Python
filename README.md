# Password Strength Checker

## Overview
This project is a Python-based Password Strength Checker that evaluates the strength of a password based on predefined security criteria.

## Features
- Checks minimum password length (8 characters)
- Verifies at least one uppercase letter
- Verifies at least one number
- Verifies at least one special character
- Displays whether the password is Strong or Weak
- Provides detailed feedback for weak passwords

## Technologies Used
- Python
- Regular Expressions (re module)

## Test Cases

### Test Case 1
Input:
Rocky

Output:
Weak Password

### Test Case 2
Input:
Rocky@

Output:
Weak Password

### Test Case 3
Input:
Rocky@258

Output:
Strong Password
